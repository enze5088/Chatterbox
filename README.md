# Chatterbox
基于BloomZ1B2 的中文语言模型。裁剪词表和WordsEmbedding后参数量为0.9B左右。使用以下开源数据进行训练。



A Chinese Language Model base on BloomZ1B.


# 数据集使用

目前主要使用Belle,alpaca_gpt4_data_zh,firefly 微调。

详细数据集收集与整理见此[介绍](./docs/datasets.md)
